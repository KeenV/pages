# animation

# animation-fill-mode

  • none：不保留样式
  • forwards：保留动画最后一帧样式。最后一帧依赖于animation-direction 和animation-iteration-count
  • backwards：一旦将动画应用于目标，立即应用第一帧，并在animation-delay期间保留该值。第一帧依赖于animation-direction
  • both：同时应用forwards与backwards
