# svg

svg元素是定义新坐标系和视口的容器。它用作SVG文档的最外层元素，但也可以用于将SVG片段嵌入SVG或HTML文档中

# Attributes

# xmlns

仅在SVG文档的最外层的svg元素上才需要此属性

# viewBox

当前SVG片段的视口坐标系

# width

svg视口的宽度

# height

svg视口的高度

# x

svg容器的x坐标，对最外层的svg元素没有影响

# y

svg容器的x坐标，对最外层的svg元素没有影响
